# 🌸 Task Calendar

A responsive and lightweight **Task Calendar** web application built using **HTML, CSS, and JavaScript**. It combines an interactive calendar with a daily to-do list, allowing users to organize tasks efficiently while enjoying multiple customizable themes.

---

## ✨ Features

* 📅 Interactive monthly calendar
* ✅ Date-wise task management
* ⏰ Add tasks with time
* 🗑️ Delete completed tasks
* 🎨 Multiple themes:

  * 🌷 Floral
  * 🍀 Nature
  * 🎌 Anime
  * 🌙 Space
* 💾 Local Storage support (tasks remain saved after refreshing the browser)
* 📱 Responsive design for desktop and mobile

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

1. Clone the repository

```bash

```

2. Open the project folder.

3. Launch **index.html** in your browser.

No additional installation or dependencies are required.
